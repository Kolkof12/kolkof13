ملخصات تعليمية ثغرات أدوات

📘 تعريف الأمن السيبراني

الأمن السيبراني هو حماية الأنظمة الرقمية، الشبكات، والبيانات من الهجمات أو الوصول غير المصرح به.

أنواع الأمن السيبراني:

  • أمن الشبكات: حماية البنية التحتية للشبكة.
  • أمن التطبيقات: تأمين البرامج والتطبيقات.
  • أمن المعلومات: حماية البيانات من التسريب أو التعديل.
  • أمن العمليات: تحديد الأذونات والسياسات التشغيلية.
  • أمن السحابة: حماية البيانات والخدمات في البيئات السحابية.

مصطلحات أساسية:

  • Firewall: جدار ناري يمنع الوصول غير المصرح به.
  • Malware: برامج خبيثة تُستخدم للهجوم أو السرقة.
  • Phishing: محاولة خداع المستخدم للحصول على معلومات حساسة.
  • Encryption: تشفير البيانات لحمايتها من التجسس.
  • Penetration Testing: اختبار الاختراق لاكتشاف الثغرات.

🛠️ أنواع الثغرات وشرحها

  • XSS (Cross-Site Scripting): تنفيذ أكواد جافاسكريبت ضارة داخل المتصفح.
  • SQL Injection: إدخال أوامر SQL ضارة في قاعدة البيانات.
  • CSRF: استغلال جلسة مستخدم لتنفيذ أوامر غير مرخصة.
  • RCE (Remote Code Execution): تنفيذ أوامر عن بُعد على الجهاز.
  • Directory Traversal: الوصول إلى ملفات غير مصرح بها عبر المسارات.

كيف تعمل الثغرات وكيف نكتشفها؟

يتم استغلال الثغرات عبر تحليل مخرجات الموقع، أو استخدام أدوات كشف مثل Burp Suite، Nmap، أو Metasploit.

🔧 أدوات Kali و Arch مع تعريفاتها

  • Nmap: أداة لفحص الشبكات واستكشاف الأجهزة المتصلة.
  • Wireshark: تحليل حركة المرور على الشبكة.
  • Metasploit: منصة قوية لاختبار الاختراق وتنفيذ الثغرات.
  • John the Ripper: كسر كلمات المرور عبر تقنيات مختلفة.
  • Aircrack-ng: اختبار اختراق شبكات الواي فاي.
  • Hydra: تنفيذ هجمات brute-force ضد الخدمات مثل SSH/FTP.
